# Portfólio Desktop Retrô

SPA de portfólio pessoal com estética Windows 95, Vue 3 e Tailwind CSS.

## Início rápido

```bash
npm install
npm run dev
```

## Personalização

Edite `src/data/portfolio.js` para alterar:

- Nome, bio e foto de perfil
- Links de redes sociais
- Experiências profissionais

## Stack

- Vue 3 (Composition API + `<script setup>`)
- Tailwind CSS 3
- Vite 6
